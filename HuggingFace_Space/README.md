## Project Overview
This project is a Streamlit-based Agent that uses PyTorch to load a trained model and help detect and diagnose if a person has diabetes.

## Features
- Interactive UI built with Streamlit
- Loads a trained PyTorch model
- Accepts user input for patient information and medical history
- Outputs the agent's diagnosis
- Lightweight and easy to deploy
